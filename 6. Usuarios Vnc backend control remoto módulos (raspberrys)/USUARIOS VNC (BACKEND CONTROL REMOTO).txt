Para acceder remotamente a los módulos raspberry y visualizarlos desde un computador desde cualquier sitio a distancia se debe instalar el programa REAL VNC VIEWER
y allí acceder con la cuenta utilizada para el proyecto iot.innovacion@lineacom.co clave: Linea2023* 
(Es necesario añadir la cuenta iot.innovacion@lineacom.co tambien en el vnc server de los raspberry ubicado entrando a estos y clickeando en la esquina superior derecha VNC server) 

Al ingresar en el computador al vnc viewer para añadir dispositivos nuevos si aún no se han enlazado nos va a pedir su respectiva direccion ip al clickear en nueva conexión,
esta se puede visualizar con el comando ifconfig entrando a la terminal de cada raspberry, una vez añadido el dispositivo con la dirección ip
este nos pedirá usuario y clave (Por defecto usuario pi y para este caso la clave Exito2020. esta fue cambiada para uno de los módulos y poder tener acceso a cada uno),
 para este caso ya están configurados y se pueden visualizar en la cuenta iot.innovacion los siguientes dispositivos RPI-01 Y RPI-02

ACCESO RPI-01
VNC: usuario: pi
clave: Exito2020.

ACCESO RPI-02
VNC: usuario: pi
clave: Exito2021.
